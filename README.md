# isBitcoinLegal

[Demo](https://hexadeciman.github.io/isbitcoinlegal/)

Data Source: [Wikipedia](https://en.wikipedia.org/wiki/Legality_of_bitcoin_by_country_or_territory)

Feel free to contribute.
